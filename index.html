📊 10년 등급 기준 (2016.03 → 2026.03)

인플레이션을 제거한 실질 CAGR의 절대 성과로 평가한다

등급기준의미
S 서울 평균 × 2 이상 (≈+8.7%/년) 압도적 수익 — 약 127개 단지 (1.2%)
A 서울 평균 이상 (≈+4.4%/년) 서울 평균 상회 — 투자가치 높다
B 기회비용(연4%) 이상 (+1.7%/년) 4% 복리 투자 우위 — 투자가치 있다
C 실질 0% 이상 인플레이션 헷지 — 실거주 마지노선
D 실질 -2% ~ 0% 실질 손실 구간
E 실질 -2% 미만 심각한 실질 손실

📈 5년 등급 기준 (2021.03 → 2026.03)

수도권 전체 단지 중 상대 순위(퍼센타일)로 평가한다

등급순위 구간설명
S상위 3%하락장 속 압도적 성과 — 약 390개 단지
A상위 3~10%우수한 성과
B상위 10~30%평균 이상
C상위 30~60%중위권
D상위 60~85%하위권
E하위 15%최하위 성과
💡 2021~2026년은 부동산 하락 이후 회복 장세다. 절대 수익보다 얼마나 잘 버텼고 먼저 회복했느냐는 상대 순위가 더 정확한 평가 기준이다.

🔄 업데이트 내역 (2026.04)

  • 📍 입지점수 시스템 신규 추가 — 100점 만점 종합 입지 평가
  • 🔍 지역 검색 순위 — 구/동 검색 시 입지점수 순위 자동 표시, 순위에서 단지 클릭 가능
  • 🎯 입지점수 필터 — 상단 필터바에 30↑~80↑ 버튼 추가
  • 🏆 단지 상세에서 동내 / 구내 입지 등수 표시
  • 🏥 대학병원 데이터 확충 (용인세브란스·인하대병원·고려대안암병원 등 추가)
  • 👮 경찰·파출소 점수 확대 (600m=4점 / 1km=3점 / 3km=2점)

📂 데이터 출처

  • 실거래가: 국토부 실거래가 공개시스템 (아파트 매매 실거래가 API)
  • 수집 기간: 2016년 1월 ~ 2026년 3월 / 서울 25구, 경기 31시군, 인천 8구
  • CPI(소비자물가지수): 통계청 — 실질 수익률 계산에 사용
  • 지도 좌표: VWorld 지오코더 API — 국토교통부 공간정보 오픈플랫폼 (공공누리 제1유형)
  • 지도 타일: VWorld 배경지도 — 국토교통부 공간정보 오픈플랫폼 (공공누리 제1유형)

❓ 데이터가 없는 경우

  • 기간 미충족: 신축 단지로 분석 기간(10년/5년)에 데이터가 없음
  • 거래 희소: 기준 시점(2016-03, 2021-03, 2026-03) 전후 거래 없음
  • 주소 미매칭: 지오코딩 실패 (단지명 특수문자 등)
  • 누락 단지 제보: 좌표 오류·주소 미매칭으로 빠진 단지는 아래 블로그 댓글로 알려주세요
